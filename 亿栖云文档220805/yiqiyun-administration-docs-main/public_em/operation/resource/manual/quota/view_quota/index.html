查看我的配额

更新时间 2020-12-01

登录管理控制台后，在主页顶部的导航栏，点击资源，然后在下拉菜单中，选择我的配额，即可查看当前账户下的资源配额。您可以查看全局以及各分区的资源配额使用情况。

在配额的详情页，查看资源后的数值，其中"/“左边的数值表示已使用的配额，”/“右边的数值表示账户所支持的总配额。

例如：1/10，1表示已使用资源，10表示资源配额。当已使用资源不足时，左边显示条会从绿色变为红色，提示您需要申请资源配额。
