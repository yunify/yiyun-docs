光盒简介

更新时间 2021-03-01

光盒，是光格网络 SD-WAN 服务的重要接入组件，便于用户最后一公里快速接入到 WAN 网，并支持公Internet 链路、专线、MPLS/VPN、4G LTE 等多种接入方式，可实时监控隧道质量，具备零部署，即插即用，动态链路切换等特性。
