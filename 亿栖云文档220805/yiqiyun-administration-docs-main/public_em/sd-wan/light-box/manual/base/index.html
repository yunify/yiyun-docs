申请光盒

更新时间 2020-12-01

使用光盒接入前，请先到管理控制台申请光盒设备。

前提条件

已获取管理控制台的账号和密码。

操作步骤

  1. 登录管理控制台。

  2. 在左上角菜单栏中，选择产品与服务 > SD-WAN > 光盒，进入光盒页面。

  3. 在左侧导航栏中，点击物理光盒，进入物理光盒页面。

  4. 点击申请光盒。

  5. 在弹出的申请光盒页面，选择光盒的型号、购买类型、租用期限，并输入您的联系信息等。

  6. 点击提交申请。

    页面将自动返回到物理光盒页面，并生成一个包含您的申请信息的订单，订单状态为审核中。

  7. 待申请审批通过后，订单状态将变更为审核已通过。

  8. 点击立即支付，请根据提示信息进行在线付款。

    付款之后，光盒管理人员会将光盒寄出，同时界面上会显示光盒状态为物流中。您可以点击查看物流，查看实时物流状态。

    收到光盒后即可开始使用，使用方法请参考快速入门、WAN 接入点。
