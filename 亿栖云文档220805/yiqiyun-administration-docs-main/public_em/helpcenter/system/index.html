    解释说明

    系统盘

    系统盘用来存储云服务器运行的操作系统，云平台为不同类型主机搭配了默认类型的系统盘。基础型云服务器默认是基础型硬盘，除基础型之外其他云服务器都搭配企业SSD型硬盘。性能指标请参见文档。

    常见问题

    1.是否可以更改系统盘容量？

    只支持增加容量，即扩容。在更改云服务器配置处操作。

    2.是否支持更换系统盘类型？

    不同类型云服务器搭配默认类型系统盘，暂不支持更换类型。
