AddResourceGroupItems

更新时间 2020-12-01

将资源加入到资源组中。

Request Parameters

Parameter name Type Description Required
zone String zone ID Yes
resource_group String 资源组 ID Yes
resources.n String 资源 ID Yes

公共参数

Example

Example Request

https://api.yiqiyun.net.cn/iaas/?action=AddResourceGroupItems
&resource_group=rg-c7hmvt8u
&zone=beta
&resources.1=i-i8wewg3l
&COMMON_PARAMS

Example Response:

{
  "action":"AddResourceGroupItemsResponse",
  "resource_group_id":"rg-c7hmvt8u",
  "resource_ids":[
    "i-i8wewg3l"
  ],
  "ret_code":0
}
