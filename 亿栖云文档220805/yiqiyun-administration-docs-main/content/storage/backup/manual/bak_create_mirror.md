## 操作场景

当备份点是基于云服务器系统盘创建时，则可以通过这个备份点制作成新镜像，基于该镜像可以创建多台和备份点状态相同的云服务器。

## 操作步骤

1. 登录管理控制台。
2. 在控制台导航栏中，选择**产品与服务** > **存储服务** > **备份**，进入**备份**页面。
3. 右键点击备份链ID号，选择**制作成新镜像**，弹出**根据云服务器备份制作镜像**页面。
4. 输入镜像名称及选择是否加密。
5. 点击**提交**。
