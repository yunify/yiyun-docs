云数据库 MongoDB 是一个开源的文档型数据库，具有高性能、高可用等优点。

云数据库 MongoDB 服务基于原生 [MongoDB](https://www.mongodb.com/) ReplicaSet 构建的云服务，提供多节点高可用架构，具备弹性扩容、容灾、备份恢复、性能优化等功能。

云数据库 MongoDB 是面向文档的 NoSQL（非关系型）数据库，数据结构由字段和值组成。
